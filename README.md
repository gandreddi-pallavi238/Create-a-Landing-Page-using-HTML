# Create-a-Landing-Page-using-HTML
# Introduction

A Landing Page is the first page that users see when they visit a website. It provides a clear overview of a product, service, or profile and encourages users to take action such as signing up or exploring more.

This project demonstrates how to create a basic landing page using HTML and CSS with proper structure and styling.

# Objective

To design a single-page landing layout

To use semantic HTML elements for better structure

To apply CSS for styling and layout

To encourage user interaction through a call-to-action

# Technologies Used

HTML5

CSS3

# Features

Semantic HTML tags like header, section, main, and footer

External CSS file for better code organization

Clean and simple user interface

Call-to-Action (CTA) button

Beginner-friendly design

# Project Structure
LandingPage/
│
├── index.html
└── style.css

Learning Outcome

Learned how to structure a webpage using semantic HTML

Understood how to link external CSS files

Improved understanding of layout and design basics

Gained experience in building a real-world beginner project
